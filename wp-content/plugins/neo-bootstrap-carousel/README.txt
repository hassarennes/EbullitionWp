== Description ==

= Are you looking for responsive, user-friendly and robust WordPress Twitter Bootstrap Carousel plugin? = 
NEO Bootstrap Carousel by <a href="http://pixelspress.com">PixelsPress</a> is an easy and light weight plugin that adds Twitter Bootstrap Carousel to your WordPress website.
You can add multiple slides and show them as a carousel on any page or post by inserting [neo_carousel_shortcode id="put post id here"] shortcode.

[Demo](http://pixelspress.com/home)

= Features =

* 100% Responsive
* Easy To Customize
* Built With Bootstrap
* Button Navigation (Bullets) (can be switched on/off, off by default) 
* Arrow Navigation (can be switched on/off)

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't even need to leave your web browser. To do an automatic install, log in to your WordPress admin panel, navigate to the Plugins menu and click Add New.

In the search field type "NEO Bootstrap Carousel" and click Search Plugins. Once you've found the plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by clicking _Install Now_.

= Manual installation =

* Download the plugin file to your computer and unzip it.
* Using an FTP program, or your hosting control panel,  upload the unzipped plugin folder to your WordPress installation's `wp-content/plugins/` directory to your web server.
* Activate the plugin through the 'Plugins' from menu in WordPress admin panel.

= Getting started =

Once installed:

1. Create a standard WordPress page or post and inside place the `[neo_carousel_shortcode id="put post id here"]` shortcode.

NEO Bootstrap Carousel plugin let you create stunning image slideshows easily in minutes. You can create your web presentations.

= NEO Bootstrap Carousel Settings =

<strong>Display Navigation</strong>

To show or hide the carousel navigation, go to the NEO Bootstrap Carousel settings and you'll see the _Display Navigation_ option, you've to select option _Yes_ If you want to show or _No_ If you want to hide.

<strong>Display Direction Arrows</strong>

To show or hide the carousel direction arrows, go to the NEO Bootstrap Carousel settings and you'll see the _Display Direction Arrows_ option, you've to select option _Yes_ If you want to show or _No_ If you want to hide.

<strong>Display Caption</strong>

To show or hide the carousel slide caption, go to the NEO Bootstrap Carousel settings and you'll see the _Display Caption_ option, you've to select option _Yes_ If you want to show or _No_ If you want to hide.

== Frequently Asked Questions ==

= Is NEO Bootstrap Carousel responsive? =

Yes, the NEO Bootstrap Carousel is responsive and displays resized image for the mobile devices and tablets.

= How to add a Carousel? =

In your WordPress admin panel, go to "NEO Bootstrap Carousel" from menu and add a new carousel. All the NEO Bootstrap Carousel lists will be shown.

= Can I add Carousel into the theme in a custom location? =

Yes, that's possible to achieve. The plugin automatically generates a shortcode for each carousel, thus you can copy and paste the shortcode for the carousel to a custom location. The shortcode is listed next to the slider published date in NEO Bootstrao Carousel > Sliders section.

= How to show carousel on the front-end? =

For operational slider, add [neo_carousel_shortcode id="put post id here"] shortcode in an existing page or add a new page and write shortcode anywhere in the page editor.

= How can you change the order of images (slides) within NEO Bootstrap Carousel? =

NEO Bootstrap Carousel uses easy drag feature. This allows to order the slides for each slider from the back end.
